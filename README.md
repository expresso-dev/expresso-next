# Next.js Setup 

## Features

* Next.js
* React 16
* Webpack 4
* ES7
* Hot Module Replacement

## Note
* if you run into errors with npm and yarn, please try using sudo 

## Installation

* clone the repo
* yarn

## Building

* npm run dev
* visit `http://localhost:3000/`

## Building & Running Production

* npm run build
* npm run start
* visit `http://localhost:3000/`
* alternatively configure /etc/hosts file
